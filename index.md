# International Keyboard Shortcuts Cheat Sheet (Mac/iPad)

> **Note**: Use `Option` = ⌥, `Shift` = ⇧, and `Command` = ⌘

## Accented Vowels

| Character | Shortcut                  |
|-----------|---------------------------|
| á         | ⌥ + e, then a             |
| é         | ⌥ + e, then e             |
| í         | ⌥ + e, then i             |
| ó         | ⌥ + e, then o             |
| ú         | ⌥ + e, then u             |
| Á, É, etc.| ⌥ + e, then ⇧ + letter    |

## Grave Accents

| Character | Shortcut                  |
|-----------|---------------------------|
| à         | ⌥ + ` (backtick), then a  |
| è         | ⌥ + `, then e             |
| ì         | ⌥ + `, then i             |
| ò         | ⌥ + `, then o             |
| ù         | ⌥ + `, then u             |

## Umlauts (Diaeresis)

| Character | Shortcut                  |
|-----------|---------------------------|
| ä         | ⌥ + u, then a             |
| ë         | ⌥ + u, then e             |
| ï         | ⌥ + u, then i             |
| ö         | ⌥ + u, then o             |
| ü         | ⌥ + u, then u             |
| Ä, Ü, etc.| ⌥ + u, then ⇧ + letter    |

## Tilde

| Character | Shortcut                  |
|-----------|---------------------------|
| ñ         | ⌥ + n, then n             |
| Ñ         | ⌥ + n, then ⇧ + n         |
| ã, õ      | ⌥ + n, then a/o           |

## Circumflex

| Character | Shortcut                  |
|-----------|---------------------------|
| â         | ⌥ + i, then a             |
| ê         | ⌥ + i, then e             |
| î         | ⌥ + i, then i             |
| ô         | ⌥ + i, then o             |
| û         | ⌥ + i, then u             |
